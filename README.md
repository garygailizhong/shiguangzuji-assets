# 时光足迹静态资源

此仓库存储时光足迹小程序的静态图片资源。

## 文件结构

```
images/

    ├── 平静.png
    ├── 思考.png

    ├── 生气.png

    ├── 惊喜.png
    ├── 焦虑.png

```

## CDN 使用

使用 jsDelivr CDN 访问：

```

```
